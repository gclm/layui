目录结构：
├─layui /layui 基础框架
│─layui_exts //存放第三方组件的目录
└─dist/ 组件的压缩后的js存在的地方
└─dtree/ 组件模块名
├─dtree.js //组件核心 JS
├─font //组件核心图标库，库中携带了dtreefont.css文件，在使用树时需要引入该文件
└─dtree.css //组件 CSS 库
使用方式：
将压缩包解压后放置在项目的任意路径，根据layui引入模块的方式将dtree.js文件引入即可使用；
在页面中，使用组件则需要引入两个class：
layui_exts/dtree/dtree.css
layui_exts/dtree/font/dtreefont.css
